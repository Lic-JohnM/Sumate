CURSOS

Aritmética

Quieres conocer como funcionan las fracciones, porcentajes y decimales. Cuando tienes una operación quieres saber cuál se opera primero, así como el uso correcto de la ley de signos, pregunta por este curso.

Álgebra

¿Quieres manejar las variables a la perfección? Usalas en expresiones algebraicas, factorizalas y operalas. Si quieres aprender estos conceptos y muchas más este curso es para ti.

Cálculo

Si viste las funciones y no las entendiste o tienes dudas de ellas, este curso es para ti.

Geometría

¿ Conoces la diferencia entre un poligono y un poliedro? ¿ Reconoces la diferencia entre área y volumen, así como cuantos metros equivalen a 1 kilometro. ? Si quieres resolver estas preguntas y muchas más, este es curso es para ti.

Trigonometría

¿Quieres manejar a la perfección el teorema de pitagoras, seno y coseno? Si deseas manejar este mundo geometrico y algebraico este es tu curso.

Estadística y probabilidad

¿ Sabes crear tablas de frecuencias y diagramas estadísticos ? ¿ Quieres recolectar información con las herramientas precisas ? Si quieres manejar la información para expresarte con ella, bienvenido al curso de estadistica.
